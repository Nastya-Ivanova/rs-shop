Кратко о приложении

Данное приложение представляет собой интернет-магазин, в котором реализован следующий функционал:

  • - единая навигация по всему магазину
  • - определение локации автоматически по IP адресу, а также возможность выбора пользователем локации из указанных городов
  • - регистрация и аутентификация пользователя
  • - поиск товаров по магазину

На главной странице реализованы два слайдера: с акционными товарами и наиболее популярными. При клике по слайдам пользователь переходит на страницу товара.

На странице «Каталог товаров» реализована навигация по всем категориям и подкатегориям товаров магазина. При клике на ссылку подкатегории пользователь попадает на страницу с карточками товаров указанной категории.

На данной странице реализована навигационная цепочка (хлебные крошки). Есть возможность сортировки товаров по популярности и цене. Изначально на странице отображается 10 товаров, по клику на кнопку «Показать больше» реализована дозагрузка товаров на страницу. Пользователь может добавить товар в Корзину, либо в Избранное. Каждая карточка товара ведет на страницу товара.

На Странице товара реализована навигационная цепочка, слайдер с автоматической сменой слайдов с изображениями товара. Указана подробная информация о самом товаре. Также есть возможность добавить товар в Корзину, либо в Избранное.

Страница «Избранные товары» содержит список добавленных пользователем в Избранное товаров. Есть возможность перейти на Страницу товара, добавить товар в Корзину, а также удалить товар из Избранного.

«Корзина» содержит список добавленных пользователем товаров. Реализована возможность изменения количества товара (уменьшение/увеличение), удаление товара из корзины.

Для оформления заказа пользователь должен заполнить форму. При заполнении формы валидной информацией размещается заказ.

Пользователь может отслеживать свои заказы на странице «Лист ожидания». На данной странице помимо сводной информации по всем заказам пользователя реализованы возможность внесения изменений в отправленный заказ и полное удаление заказа.
